# daign-dom-pool

[![NPM package][npm]][npm-url]

Helper for reusing dom nodes.

## Installation

```sh
npm install @daign/dom-pool --save
```

## Scripts

#### Build

    npm run build

#### Run lint analysis

    npm run lint

#### Run unit tests with code coverage

    npm run test
